What is Included in the Free Trial?

  • Software Functionality
  • Data
  • What Data Is Real-Time
  • Support
  • Is Historical and Real-time Futures Data Included? (Opens a new page)

Software Functionality

[Link] - [Top]

When you create a Sierra Chart account, you can request a Free trial period of 15 days.

If you have been unable to use this trial time, then contact Sierra Chart Account Support about this.

During the Sierra Chart free trial period you get unrestricted access to all of the software functionality that is available.

With the trial you can also use any of the supported Trading Services provided by Sierra Chart.

If a Trading service, is an external supported service like Interactive Brokers, CQG, Rithmic, and others, these are not supported in a free trial. You will need to change to a paid Service Package for these. After changing the Service Package, you do not have to make a payment until the usage time expires.

As an alternative during a free trial, use the Simulated Futures Trading Service.

Data

[Link] - [Top]

The Sierra Chart free trial includes access to the following Sierra Chart provided Data Services listed below.

  • Delayed Exchange Data Feed: Provides streaming delayed data for many futures and stock exchanges.
  • Sierra Chart Real-time and Historical Forex/CFD Data Service
  • Sierra Chart Historical Daily and Intraday Data Service
  • Sierra Chart Market Statistics Data Feed
  • Cryptocurrency Data Services

The above listed services are not included in what is called Free Mode, which is the mode you are placed in after your free functional trial is completed. After your free trial is completed, you will need to make a payment for Sierra Chart usage time.

With the trial you can also use any of the supported Trading Services provided by Sierra Chart. If a Trading service, is an external supported service like Interactive Brokers, CQG, Rithmic, and others, these are not supported in a free trial. You will need a paid account for these. As an alternative during a free trial, use the Simulated Futures Trading Service.

If you want real-time futures data, then refer to the Is Historical and Real-time Futures Data Included? page for complete information about this subject.

What Data Is Real-Time

[Link] - [Top]

To know what categories of symbols that real-time data is provided for directly with the Sierra Chart Service Packages, refer to the Real-time Data page.

Support

[Link] - [Top]

For the available support options, go to the Support page. Keep in mind that free trial users do not have entitlement to support other than account issues, pricing questions, problems with basic operation or basic questions.

Advanced support is not provided to free trial users.

If you need clarification or further help on any of this, use the appropriate Support option.
