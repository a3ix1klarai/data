Global Graphic Settings

[Link] - [Top]

This section explains opening the Global Graphics Settings window for the various Colors, Line Styles, and Line Widths settings described on this page.

Select Global Settings >> Graphics Settings on the menu to customize the Fonts, Colors, Line Styles, and Line Widths globally. The relevant settings and the descriptions are described below.

Chart and Trade DOM Specific Graphic Settings

[Link] - [Top]

This section explains opening the Chart / Trade DOM Graphics Settings window for the various Colors, Line Styles, and Line Widths settings described on this page.

To customize the Fonts, Colors, Line Styles, and Line Widths for a specific chart or Trade DOM, select Chart >> Graphics Settings on the menu.

Make certain Use Global Graphics Settings Instead of These Settings is unchecked in this case. The relevant settings and the descriptions are described below.

Font Settings

[Link] - [Top]

Depending upon whether the chart or Trading DOM is using either the global Graphics Settings or its own Graphics Settings, you will open either the Global Graphics Settings window through Global Settings >> Graphics Settings, or the chart specific Graphics Settings window through Chart >> Graphics Settings to modify the primary font of the chart or Trade DOM.

For further details, refer to Graphics Settings. The chart specific Graphics Settings window is only used if the Use Global Graphics Settings Instead of These Settings is disabled on that window.

Through either of those settings Windows, select the Fonts tab. In the list fonts select Chart Text. Press the Select button. Adjust the font properties and press OK.

To modify the font for the depth of market quantity/size numbers on the Chart DOM, in the list of fonts select DOM Quantities/Sizes. Press the Select button. Adjust the font properties and press OK.

Chart Trading and Chart DOM Colors, Line Styles, and Line Widths

[Link] - [Top]

Chart DOM Column Separator Lines

[Associated Settings Window] - [Link] - [Top]

When enabled, this specifies the color, width, and line style of the vertical bars that separate the columns in the Chart Trading DOM. It is necessary to check the Enable option for the columns to be drawn.

Chart DOM Buy Column Background

[Associated Settings Window] - [Link] - [Top]

When enabled, specifies the background color of the Buy column. This option is also used when the Bid/Ask columns are combined, and when the Buy/Bid columns are combined. It is necessary to check the Enable option, for the background color specified with this color setting to actually be drawn.

Chart DOM Buy Column Lines (also Order Entry Highlight Box)

[Associated Settings Window] - [Link] - [Top]

The color of the vertical Buy order column lines and the box which highlights the price level being pointed to in the Buy column.

In the case of the box which highlights the price level, this is drawn with a special drawing mode for efficiency reasons, so it may not match exactly this color setting. The Chart DOM Buy Column Background color will affect the highlight box color.

Chart DOM Sell Column Background

[Associated Settings Window] - [Link] - [Top]

When enabled, specifies the background color of the Sell column. This option is also used when the Sell/Ask columns are combined. It is necessary to check the Enable option, for the background color specified with this color setting to actually be drawn.

Chart DOM Sell Column Lines (also Order Entry Highlight Box)

[Associated Settings Window] - [Link] - [Top]

In the case of the box which highlights the price level, this is drawn with a special drawing mode for efficiency reasons, so it may not match exactly this color setting. The Chart DOM Sell Column Background color will affect the highlight box color.

[Associated Settings Window] - [Link] - [Top]

Chart DOM Ask Column Background

When enabled, specifies the background color of the Ask column. It is necessary to check the Enable option, for the background color specified with this color setting to actually be drawn.

This is the color of the connecting lines when Global Settings >> Chart Trade Settings >> Order Fills >> Color Connecting Lines Based on Profit/Loss is disabled in the Chart Trade Settings window.

Chart Trading Order Fill Entry/Exit Connecting Lines Secondary

[Associated Settings Window] - [Link] - [Top]

This sets the Color, Line Style and Width of Entry and Exit order fill connecting lines.

This setting is only relevant when Displaying Order Fills on the chart.

This is the color of the connecting line when the connecting line is part of a losing trade. This color setting is only used if Global Settings >> Chart Trade Settings >> Order Fills >> Color Connecting Lines Based on Profit/Loss is enabled in the Chart Trade Settings window.

Connecting lines can be enabled by selecting Global Settings >> Chart Trade Settings. Enable the Order Fills >> Display Entry/Exit Connecting Lines option.

To determine if a trade is a losing trade and therefore use the secondary color, it is necessary to enable the Global Settings >> Chart Trade Settings >> Order Fills >> Display Trade Profit Loss option in order to see the P/L: value on the exit fill displayed on the chart.

If P/L: displays a negative value, then this indicates it is a losing trade. Otherwise, it is a profitable trade.

Also refer to Chart Trading Order Fill Entry/Exit Connecting Lines Primary.

Trade DOM Position Line Long/Profit

[Associated Settings Window] - [Link] - [Top]

This sets the Color, Line Style, and Draw Width of a Trade Position line when the Trade Position for the symbol of the chart is Long and the Trade Position line is set to be colored based upon whether the Trade Position is Long or Short, or the Trade Position has a Profit and the Trade Position line is set to be colored based upon whether there is a Profit or Loss.

To control whether coloring is based upon Long/Short or Profit/Loss, set Global Settings >> Chart Trade Settings >> Position >> Coloring of Position Line Based On to either Long/Short or Profit/Loss.
