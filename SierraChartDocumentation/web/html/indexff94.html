Real-Time Data

  • Introduction

Introduction

[Link] - [Top]

This page documents the subject of real-time market data.

It explains what categories of data have real-time data which are included with the Sierra Chart Service Packages.

For other categories of data that the Service Packages do not directly include real-time data for, it explains how to obtain that data.

Historical Data is provided for all symbols.

There is also streaming delayed data available in Sierra Chart for futures markets. While this data is not real-time, it still is a streaming data which also includes market depth data.

What Real-Time Data is Directly Included with the Service Packages

[Link] - [Top]

Refer to the image below to know what particular symbols provide real-time streaming data as part of the Sierra Chart Service Packages. This information applies to trials and also to the paid Service Packages. The listed symbols have real-time data directly provided by the Service Packages.

Regarding the red highlighted sections in the screenshot below, refer to the notes in the above screenshot for a description.

So the categories which include real-time data as part of the Service Packages are as follows:

  • CFD
  • Cryptocurrencies (Binance, Deribit, BitMex, and others)
  • Forex
  • FX Indexes
  • Market Stats
  • CME, CBOT, NYMEX, COMEX, EUREX Exchange Futures (streaming delayed data)

Delayed streaming data is provided for CME, CBOT, NYMEX, COMEX and EUREX exchange symbols.

Real-time Futures, Equities/Stocks, Options and Cash Index Data

[Link] - [Top]

To receive real-time streaming data for Futures, Equities/Stocks, Options or Cash indexes which come from exchanges, you will need to connect Sierra Chart to an external supported Data and Trading service which provides the particular data that you require from the particular exchanges.

Refer to the supported Supported Data and Trading Services page for the list and complete details. Each supported Data and Trading service will only provide certain types of data and from certain exchanges. For example, a Trading service that supports futures trading is not going to provide equities/stocks or cash index or options data.

Real-time data from exchanges in almost all cases has an additional cost.

Or, to receive real-time streaming data for Futures, Equities/Stocks, Options or Cash indexes from exchanges, you can pay for and activate one of the Real-Time Exchange Data Feeds Available from Sierra Chart. These are Sierra Chart provided data feeds and are an additional cost to the Service Packages.

For a complete understanding of real-time futures data, refer to the Is Historical Real-time Futures Data Included page. This is an essential read. Be sure to read the page Is Historical Real-time Futures Data Included completely before you ask further questions about real-time futures data.
