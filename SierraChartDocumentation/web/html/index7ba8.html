Instructions

[Link] - [Top]

Follow these steps to translate the Sierra Chart menus to the desired language.

  1. Update Sierra Chart to the current version.
  2. Select Global Settings >> Customize Menu Items.
  3. From the Available Preconfigured Customizations list, select the particular language you want to use.
  4. Press the Set button.
  5. Press the OK button.
  6. Restart Sierra Chart for a thorough translation of all menu items.

Individual menu items can be customized any way that you require following the Customize Menu Items instructions.

Revert To English

[Link] - [Top]

Follow these steps to revert the Sierra Chart menus to English.

  1. Select Global Settings >> Customize Menu Items.
  2. Press the Set All to Defaults button.
  3. Press the OK button.
  4. You may need to restart Sierra Chart for a thorough setting of all menu items to default.

Generating the DefaultMenuCustomizations.xml File

[Link] - [Top]

It is possible to generate a default menu customizations XML file from within Sierra Chart to allow the editing of each and every menu command within the program with a custom name. This allows for easy direct editing of an XML. Follow the steps below:

  1. Select General Settings >> Customize Menu Items.
  2. Press on the Write Default Menu Cutomizations to File button.
  3. Press OK to close the dialog.
  4. The file will be saved to the Sierra Chart program folder, the location will also be shown within the Message Log.
