Sierra Chart - Image Gallery

This is the public chart image gallery. To share a chart image, select Edit >> Screen Images >> Upload Chart on the Sierra Chart menu to upload a chart. Select Help >> Account Management. Click on Manage Images. Locate the chart image among the images you have uploaded. Add a note to it if you wish. And click on Share Image in the Public Image Gallery. That's it!
