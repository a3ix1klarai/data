Introduction

[Link] - [Top]

Sierra Chart has multiple Service Packages which are explained below.

  • You can Activate a particular Service Package on your account.
  • You can upgrade your Sierra Chart Service Package at any time.
  • You can also evaluate the highest available Service Package if your account is not currently on this Service Package.

Refer to the Free Trial Help Topic to see what is included in the trial.

Be sure to read about possible additional expenses for data.

Packages

[Link] - [Top]

Base Packages

[Link] - [Top]

The Base Packages from Sierra Chart do not support connections to external provided Data or Trading services. For example, they will not connect to Interactive Brokers, CQG or Rithmic.

These Base Packages support connectivity to Sierra Chart provided Data and Trading services only. However, they do not support connectivity to the Denali Exchange Data Feed.

In order to connect to any external Data or Trading service or to the Denali Exchange Data Feed, one of the Integrated Packages (Package 10, 11, or 12) must be used.

The following data feeds or simulated trading services can be used with these Base Packages:

  • Sierra Chart FOREX and CFD Data Service
  • Real-Time data enabled through the Sierra Chart Exchange Data Feed
  • Sierra Chart Market Statistics Data Feed
  • Cryptocurrency / Bitcoin Data and Trading Services
  • Delayed Streaming Exchange Data Feed
  • Simulated Trading Service

Integrated Packages

[Link] - [Top]

The Integrated Packages from Sierra Chart includes external connectivity from Sierra Chart to any of the Supported Data and Trading Services.

In addition, these packages can be used with any of the Sierra Chart data feeds, including the real-time Denali Exchange Data Feed.

[Link] - [Top]

Price: 36 USD/month

The Integrated Standard Sierra Chart Service Package includes external connectivity along with all functionality of the software except what is listed in the Integrated Sierra Chart with Advanced Features section and the Market by Order data.

Sierra Chart with Advanced Features and Market by Order (Service Package 12)

Included Data and Simulated Trading Services

[Link] - [Top]

The following data and services are available with any Sierra Chart package:

  • Sierra Chart Historical Data Service
  • Sierra Chart Real-time and Historical Forex/CFD Data Service
  • Sierra Chart Market Statistics Data Feed
  • Cryptocurrency / Bitcoin Data and Trading Services
  • Delayed Exchange Data Feed. High quality and detailed, streaming delayed and historical data. With market depth data and market by order data.
  • Simulated Trading Service. You can perform unlimited simulated trading, using real time or delayed data using a server-based environment.

External Supported Services

[Link] - [Top]

External Services are external Data or Trading services. They are not included in the Sierra Chart Service Package price since they are externally provided functionality.

For a list of the services which are externally provided, refer to Supported Data and Trading Services. In that list, only the services which are identified as "External" are considered external services. The others are considered included services or separately paid for Data services.

An example of an external service would be Interactive Brokers, CQG or Rithmic.

How Much Does the Sierra Chart Charting and Trading Software Cost plus Data?

[Link] - [Top]

The price of the Sierra Chart software and data for Sierra Chart depends upon what Data or Trading service you will be using Sierra Chart with and what Service Package you choose.

Sierra Chart supports many external Data and Trading services, including our own Trading and Data services.

The external services are separate from us. Data services provided with the Service Packages include the Sierra Chart Historical Data Service, Sierra Chart Real-Time and Historical Forex and CFD Data Service, and the Sierra Chart Market Statistics Data Feed.

If the service is separate from us, then there potentially is an additional charge for the data. Check with the Trading service you use to find out the monthly cost. In the case of futures data, whether you use a Trading service for the market data, an external Data service, or the Denali Exchange Data Feed (described below), you will have to pay exchange fees.

There are also the Real-Time Exchange Data Feeds Available from Sierra Chart which are Data services provided by Sierra Chart that provide real-time futures, stocks and cash indices data. These data services have their own separate pricing from the Service Packages described on this page, and therefore are an additional cost.

For information about real-time futures data, refer to Futures Data.

For further information and pricing for external services, refer to the Data and Trading Services page.

EUREX Historical and Real-time Data

[Link] - [Top]

As a matter of policy, EUREX historical and real-time data is provided by the Denali Exchange Data Feed when using Sierra Chart. If only historical data is required for EUREX, then this is provided by the Sierra Chart Historical Data Service.

These are the only recommended sources of data for EUREX data. We do not recommend using any other source. And no support will be provided for using EUREX data from any other source.

It is possible to access data from other sources, but no technical help is provided and you must be using Service Packages 10, 11, or 12. To change your Service Package, refer to Services Activation.

One Time Purchase Option?

[Link] - [Top]

Occasionally we get asked if we offer a one-time purchase option for Sierra Chart. The answer is no, and this is something that would never be offered.

It is inconsistent/incompatible with the services that we offer. Sierra Chart is not merely software, but provides extensive market data services, order routing services to exchanges, support services and continuous updates to our software which is necessary for it to work with external Data/Trading services due to various changes with those services which occur on an ongoing basis.

We also continuously update the software with various improvements and new features.

Therefore, our costs must be covered with continuous payments for the software and services.
