Using Sierra Chart with Ninja Trader Brokerage

  • Overview

Overview

[Link] - [Top]

Often we are asked by Ninja Trader brokerage customers or Tradeovate brokerage customers about using Sierra Chart, or in addition to using Sierra Chart, using the Denali Exchange Data Feed.

Sierra Chart's position is that for Ninja Trader and Tradeovate brokerage accounts, is that if you are trading the CME Group of markets, you should use our Teton Order Routing Service and change to a broker supporting this service.

If you want to use Sierra Chart for trading, you will subscribe to Sierra Chart and use the transaction free Teton Order Routing Service for CME Group trading, and a broker supporting the Teton service.

If you want to use Sierra Chart just for charting, then you will subscribe to Sierra Chart and also use a Service Package that includes the Denali Exchange Data Feed and perform the set up for this data feed.

In this last case for charting only, if you require real-time data from the CME Group of exchanges at the lower priced nonprofessional exchange fees, then follow the Easy Solution to CME Funded Trading Account Requirement. With this method you can open an account for 100 USD at a different broker to authorize the real-time CME data at the nonprofessional fees.

It is not possible to use your Ninja Trader or Tradeovate accounts to authorize the CME Group data at the lower priced nonprofessional exchange fees, for technical connectivity reasons.

The use of the Denali Exchange Data Feed and the Teton Order Routing Service, will provide the best quality and trouble-free services to our users and allow us to support you well.

Finally we will provide no technical help at all to use the technical systems used by Ninja Trader or Tradeovate. They will not receive support by Sierra Chart and as we understand the connectivity to them is not supported anyway by Ninja Trader.
