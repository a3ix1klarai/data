Command Line Parameters

[Link] - [Top]

Command line parameters are used when launching Sierra Chart. This can be done from a command line, a shortcut, or through other methods.

List of Command Line Parameters

[Link] - [Top]
Parameter Description
/Service:[service]

Sets the Data or Trading service to use. [service] is an internal identifier for the service. These are undocumented.
/ServiceAccount:[trade account]

Sets the Trading service trading account identifier.
/ServiceUsername:[username]

Sets the Data or Trading service username.
/ServicePassword:[password]

Sets the Data or Trading service password.
/SCAccountName:[Sierra Chart account name]

Sets the Sierra Chart Account Name to be used to login to Sierra Chart.
/SCPassword:[Sierra Chart password]

Sets the Sierra Chart Password to be used to login to Sierra Chart.

Example

[Link] - [Top]
SierraChart.exe /service:transact.trading /serviceaccount:1234

Using Custom Program Icon File

[Link] - [Top]

It is supported in Sierra Chart to use your own icon file which is displayed at the top left of the main window in any detached charts.

This is simply supported by creating an icon file, with the name Program.ico and placing it into the main Sierra Chart folder which typically is C:\SierraChart.
