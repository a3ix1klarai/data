Tools Menu

All Chart Drawing tools and Chart Drawing modification tools are on the Tools menu.

Refer to the Chart Drawing Tools documentation page for complete information about drawing tools and all of the commands found on this menu.

Any commands on the Tools menu which are not documented on the Chart Drawing Tools page are documented below.

More Tools >> Log Drawing Statistics (Tools menu)

[Link] - [Top]

This command when selected will add to the Window >> Message Log, the number of drawings of each category and the total drawing time for those in milliseconds. Below is an example for the user drawn chart drawings:

(Symbol) #1 | User Drawn Chart Drawings Draw Time in Milliseconds: 0. Count: 0
